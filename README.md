# 🗺️ U.S. States Guessing Game (Python Turtle)

An interactive Python game that helps users learn and memorize all 50 U.S. states using a visual map interface.

---

## 🎯 Project Overview

This project is built using Python’s **turtle graphics** and **pandas**.
Users are prompted to guess U.S. states, and correct answers are displayed directly on the map.

💡 The game also provides **visual feedback** by showing missed states in **red** when the player exits.

---

## 🚀 Features

* 🧠 Interactive learning experience
* 🗺️ Visual map-based gameplay using Turtle
* ✅ Correct guesses appear in their correct geographic positions
* ❌ Missed states are displayed in **red** for instant feedback
* 📊 Progress tracking (score out of 50)
* 🔄 Continuous input until all states are guessed or user exits

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/us-states-game.git
```

2. Navigate to the project folder:

```
cd us-states-game
```

3. Install dependencies:

```
pip install pandas
```

4. Run the game:

```
python main.py
```

---

## 🎮 How to Play

* Enter the name of any U.S. state
* If correct → it appears on the map
* Keep guessing until you reach all 50 states
* Type **"Exit"** to end the game early

👉 When exiting:

* All missed states will appear on the map in **red**
